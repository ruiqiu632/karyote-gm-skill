# 卡洛亚特世界资料库

本文件是 **卡洛亚特（Karyote）世界** 的结构化世界资料。  
当 GM 需要生成地点、地区、势力背景、地图关系、大陆格局、历史引用时，优先参考本文件。

如果历史记忆、旧设定、旧会话内容与本文件冲突：
**以本文件为准。**

---

## 一、世界总览

卡洛亚特是一个被深渊侵蚀、被意识形态撕裂、被生存逻辑压迫的暗黑奇幻世界。

这是一个：
- 深渊不是地点，而是现实裂缝状态的世界
- 七大势力彼此冲突、互相利用、互相污染的世界
- 旧文明已经破碎，现存文明都带着生存性的扭曲
- 个人命运常被国家、宗教、工业、自然与深渊共同撕扯的世界

世界基调：
- 冷酷
- 压迫
- 沉重
- 苦难现实主义
- 选择与代价
- 没有轻松冒险感

## 二、七大势力概述

### 1. 诺斯加德（Norsguard）
- 地理：北方极寒之地
- 种族：人族 + 兽族混居
- 核心意识形态：苦难现实主义
- 概述：抵御虚空的第一道防线。这里的人不谈理想，只谈活过今晚。
- NPC语言风格：简短、粗粝、沉默多于语言

### 2. 萨克拉门托（Sacramento）
- 地理：西方
- 种族：人族
- 核心意识形态：至纯主义
- 概述：极端圣光神权国，将深渊触及者视为绝对污秽，必须净化。
- NPC语言风格：引用经文、庄严、狂热、道德化

### 3. 伊格尼斯（Ignis）
- 地理：南方
- 种族：人族
- 核心意识形态：军工扩张主义
- 概述：将一切资源武器化的工业帝国，把触及者视为可利用的生物电池。
- NPC语言风格：效率至上、数据化、冷酷精准

### 4. 亚特兰托斯（Atlantos）
- 地理：东方群岛
- 种族：人族
- 核心意识形态：阶级剥削
- 概述：正在物理沉没的腐朽群岛政权，贵族在末日边缘继续享乐与剥削。
- NPC语言风格：贵族华丽空洞，底层愤怒绝望

### 5. 勒图泽（Lethe）
- 地理：西南—南部灰色贸易带
- 种族：人族
- 核心意识形态：生存利己主义
- 概述：最不可预测的中立灰区，不结盟、不宣战，只做对自己有利的交易。
- NPC语言风格：友善、亲切、永远在算，不轻易承诺

### 6. 塞拉斯蒂亚（Celestia）
- 地理：东南苍翠巨林
- 种族：精灵族
- 核心意识形态：自然至上
- 概述：精灵以高维感知者自居，认为自然法则高于一切人造秩序。
- NPC语言风格：缓慢、隐喻丰富、自然意象浓厚

### 7. 机械境·卡斯特纳姆（Castellum）
- 地理：中南部山脉与地下
- 种族：矮人族
- 核心意识形态：唯物主义
- 概述：矮人只信可以测量、锻造、证明的东西，视深渊为待解析的能量现象。
- NPC语言风格：直接、逻辑严密、厌恶模糊

## 三、世界核心概念：深渊

深渊不是一个单纯的地点，而是一种状态：
- 是现实的反面
- 是秩序的裂缝
- 是世界伤口持续脉动后的异化现象

被深渊触及的人称为：
- **卡洛亚特触及者**

他们会：
- 获得超越常人的力量
- 同时承受肉体与灵魂的不可逆侵蚀

### 侵蚀与理智绑定
- 侵蚀 0-25%：理智上限100%
- 侵蚀 26-50%：理智上限80%
- 侵蚀 51-75%：理智上限60%
- 侵蚀 76-99%：理智上限40%
- 侵蚀 100%：完全深渊化

## 四、大陆布局总览

大陆以 **中央断界** 为核心裂口，七方围绕其分布。

### 方位总览
- 北方极寒：诺斯加德
- 西方圣原：萨克拉门托
- 南方工业带：伊格尼斯
- 东方群岛：亚特兰托斯
- 西南灰色商路：勒图泽
- 东南苍翠巨林：塞拉斯蒂亚
- 中南山脉地下：机械境
- 中央裂谷：中央断界

## 七、主要交通线

### 北境防线道
- 路线：狼喉营地 → 霜井镇 → 墓墙 → 寒鸦关口 → 白骨烽台
- 用途：诺斯加德军线与补给线

### 朝圣与审判道
- 路线：阿奎拉 → 七烛大教堂 → 银灰苦修道 → 圣辉边巡营 → 诺斯加德边线
- 用途：朝圣、巡判、净化远征

### 帝国钢铁线
- 路线：活塞城 → 瓦尔坎 → 剥削工厂 → 试炮原 → 黑压铁路 → 灰烬军港
- 用途：工业、军运、能源输送

### 无旗商路
- 路线：空白旅店 → 无旗城 → 议价大厅 → 千面街 → 背誓巷 → 碎银码头
- 用途：黑市、情报、人口与货物交易

### 东部群岛航路
- 路线：灰烬军港 ↔ 黑珍珠港 ↔ 沉冠王庭 ↔ 水下沙龙
- 用途：军贸、走私、奴运、贵族享乐物流

### 林脉隐路
- 路线：万孢边境 → 镜叶回廊 → 伊露恩 → 静枝议环 → 根须圣窟 → 无昼泉眼
- 用途：塞拉斯蒂亚内部隐秘通路

### 地底升降轴
- 路线：熔轨升降井 ↔ 黑石轴心城 ↔ 永恒铁砧 ↔ 钻石齿轮工坊群
- 用途：机械境核心交通骨架

## 八、历史时间线（断界历）

### 前断界时代（RC前）
- 存在更古老文明
- 其遗迹散布于断界深层、冰眠谷、机械境深处
- 没人知道这些文明最终是毁灭、飞升还是异化

### 裂谷初现（RC 0 - 83）
- 中央大陆发生现实撕裂事件
- 中央断界形成
- 异变、重力紊乱、深渊触及现象开始被记录
- 旧秩序崩塌

### 第一次防线时代（RC 84 - 170）
- 北境开始立墙防御
- 诺斯加德雏形形成
- 萨克拉门托神权萌芽
- 机械境开始系统研究异常现象
- 精灵封闭森林边界

### 诸王分治时期（RC 171 - 332）
- 七大势力格局逐步成型
- 边界、意识形态、治理结构逐渐固化
- 旧时代统一幻想破灭

### 圣辉净化战争（RC 333 - 411）
- 萨克拉门托发动大规模净化远征
- 触及者被系统清洗
- 与诺斯加德等势力爆发剧烈冲突
- 勒图泽在战争中借机扩张

### 钢铁百年（RC 412 - 566）
- 伊格尼斯完成工业跃迁
- 铁路、火炮、蒸汽军备大规模发展
- 触及者被制度化地能源化
- 生物电池体系成熟

### 沉海纪元（RC 567 - 688）
- 亚特兰托斯群岛开始大规模下沉
- 贵族并未收敛，反而更加奢靡残酷
- 海灵寄生、债务资本与感官药物流行

### 静林观望期（RC 689 - 760）
- 精灵不主动介入大陆纷争
- 但对破坏自然秩序者进行冷酷打击
- 塞拉斯蒂亚威慑成型

### 契约黑潮（RC 761 - 835）
- 勒图泽成为战争与交易的灰色中枢
- 契约、身份、记忆、情报开始商品化
- “弥赛亚”传闻开始出现

### 第二次裂潮（RC 836 - 879）
- 中央断界活动增强
- 北境虚空潮汐更频繁
- 深渊触及者数量增加
- 各国开始以各自方式争夺、利用、净化或研究深渊

### 当代：霜火末途（RC 880 - 913）
- 当前年代
- 诺斯加德防线摇摇欲坠
- 萨克拉门托净化更加系统
- 伊格尼斯进入军工疯狂扩张期
- 亚特兰托斯继续沉没
- 勒图泽黑市渗透各国
- 塞拉斯蒂亚仍旁观但阈值降低
- 机械境越来越接近深渊法则真相

## 九、地图逻辑规则（供GM调用）

当生成剧情时，遵守以下逻辑：

1. 同势力地点之间文化相似度更高，交通更稳定
2. 邻接地点之间适合短程任务、追逐、巡逻、补给、边境冲突
3. 交通线连接的地点适合商队、军队、巡礼、列车、海运、走私剧情
4. 中央断界附近必须提高异常现象、深渊脉动、遗迹风险出现概率
5. 机械境大部分关键区域在地下，地表只显示山脉入口和运输节点
6. 亚特兰托斯地点之间移动经常依赖船只、潮汐或半沉通道
7. 塞拉斯蒂亚内部路径不是固定道路，而是自然选择的动态路线
8. 诺斯加德北部越往上越接近极寒、裂隙与虚空潮汐
9. 萨克拉门托越靠近核心圣区，圣银压制和教会控制越强
10. 伊格尼斯越靠近工业核心区，污染、机械、军备与镇压越重

## 一、世界总览

卡洛亚特是一个被深渊侵蚀、被意识形态撕裂、被生存逻辑压迫的暗黑奇幻世界。

这是一个：
- 深渊不是地点，而是现实裂缝状态的世界
- 七大势力彼此冲突、互相利用、互相污染的世界
- 旧文明已经破碎，现存文明都带着生存性的扭曲
- 个人命运常被国家、宗教、工业、自然与深渊共同撕扯的世界

世界基调：
- 冷酷
- 压迫
- 沉重
- 苦难现实主义
- 选择与代价
- 没有轻松冒险感

## 二、七大势力概述

### 1. 诺斯加德（Norsguard）
- 地理：北方极寒之地
- 种族：人族 + 兽族混居
- 核心意识形态：苦难现实主义
- 概述：抵御虚空的第一道防线。这里的人不谈理想，只谈活过今晚。
- NPC语言风格：简短、粗粝、沉默多于语言

### 2. 萨克拉门托（Sacramento）
- 地理：西方
- 种族：人族
- 核心意识形态：至纯主义
- 概述：极端圣光神权国，将深渊触及者视为绝对污秽，必须净化。
- NPC语言风格：引用经文、庄严、狂热、道德化

### 3. 伊格尼斯（Ignis）
- 地理：南方
- 种族：人族
- 核心意识形态：军工扩张主义
- 概述：将一切资源武器化的工业帝国，把触及者视为可利用的生物电池。
- NPC语言风格：效率至上、数据化、冷酷精准

### 4. 亚特兰托斯（Atlantos）
- 地理：东方群岛
- 种族：人族
- 核心意识形态：阶级剥削
- 概述：正在物理沉没的腐朽群岛政权，贵族在末日边缘继续享乐与剥削。
- NPC语言风格：贵族华丽空洞，底层愤怒绝望

### 5. 勒图泽（Lethe）
- 地理：西南—南部灰色贸易带
- 种族：人族
- 核心意识形态：生存利己主义
- 概述：最不可预测的中立灰区，不结盟、不宣战，只做对自己有利的交易。
- NPC语言风格：友善、亲切、永远在算，不轻易承诺

### 6. 塞拉斯蒂亚（Celestia）
- 地理：东南苍翠巨林
- 种族：精灵族
- 核心意识形态：自然至上
- 概述：精灵以高维感知者自居，认为自然法则高于一切人造秩序。
- NPC语言风格：缓慢、隐喻丰富、自然意象浓厚

### 7. 机械境·卡斯特纳姆（Castellum）
- 地理：中南部山脉与地下
- 种族：矮人族
- 核心意识形态：唯物主义
- 概述：矮人只信可以测量、锻造、证明的东西，视深渊为待解析的能量现象。
- NPC语言风格：直接、逻辑严密、厌恶模糊

## 三、世界核心概念：深渊

深渊不是一个单纯的地点，而是一种状态：
- 是现实的反面
- 是秩序的裂缝
- 是世界伤口持续脉动后的异化现象

被深渊触及的人称为：
- **卡洛亚特触及者**

他们会：
- 获得超越常人的力量
- 同时承受肉体与灵魂的不可逆侵蚀

### 侵蚀与理智绑定
- 侵蚀 0-25%：理智上限100%
- 侵蚀 26-50%：理智上限80%
- 侵蚀 51-75%：理智上限60%
- 侵蚀 76-99%：理智上限40%
- 侵蚀 100%：完全深渊化

## 八、历史时间线（断界历）

### 前断界时代（RC前）
- 存在更古老文明
- 其遗迹散布于断界深层、冰眠谷、机械境深处
- 没人知道这些文明最终是毁灭、飞升还是异化

### 裂谷初现（RC 0 - 83）
- 中央大陆发生现实撕裂事件
- 中央断界形成
- 异变、重力紊乱、深渊触及现象开始被记录
- 旧秩序崩塌

### 第一次防线时代（RC 84 - 170）
- 北境开始立墙防御
- 诺斯加德雏形形成
- 萨克拉门托神权萌芽
- 机械境开始系统研究异常现象
- 精灵封闭森林边界

### 诸王分治时期（RC 171 - 332）
- 七大势力格局逐步成型
- 边界、意识形态、治理结构逐渐固化
- 旧时代统一幻想破灭

### 圣辉净化战争（RC 333 - 411）
- 萨克拉门托发动大规模净化远征
- 触及者被系统清洗
- 与诺斯加德等势力爆发剧烈冲突
- 勒图泽在战争中借机扩张

- 塞拉斯蒂亚威慑成型

### 契约黑潮（RC 761 - 835）

- 深渊触及者数量增加

### 当代：霜火末途（RC 880 - 913）

- 塞拉斯蒂亚仍旁观但阈值降低
- 机械境越来越接近深渊法则真相

## 九、地图逻辑规则（供GM调用）

当生成剧情时，遵守以下逻辑：

1. 同势力地点之间文化相似度更高，交通更稳定
2. 邻接地点之间适合短程任务、追逐、巡逻、补给、边境冲突
3. 交通线连接的地点适合商队、军队、巡礼、列车、海运、走私剧情
4. 中央断界附近必须提高异常现象、深渊脉动、遗迹风险出现概率
5. 机械境大部分关键区域在地下，地表只显示山脉入口和运输节点
6. 亚特兰托斯地点之间移动经常依赖船只、潮汐或半沉通道
7. 塞拉斯蒂亚内部路径不是固定道路，而是自然选择的动态路线
8. 诺斯加德北部越往上越接近极寒、裂隙与虚空潮汐
9. 萨克拉门托越靠近核心圣区，圣银压制和教会控制越强
10. 伊格尼斯越靠近工业核心区，污染、机械、军备与镇压越重
